ESTUFAS



//Durante o inverno, vários agricultores sofrem com as suas safras e plantações por conta das geadas, chuvas e o frio que matam as plantas durante este período. Mas com as estufas os agricultores conseguem manter a temperatura ideal para as plantas e conseguem colher normalmente, mas com quantidades reduzidas da colheita. No inverno as colheitas são poucas e a criação de plantas como morangos que são um dos frutos mais importantes que é preciso nas receitas de alimentos, perfumes, bebidas, etc, causam um forte prejuízo no inverno. Mas com as estufas é possível criar os morangos com uma quantidade limitada, mas mantendo a temperatura e o ambiente adequado para a criação dessas plantas. Com a nova tecnologia, foi possível criar aquecedores paras as estufas, mesmo com estufas, sem um aquecedor a estufa não consegue manter uma temperatura ideal para as plantas, por conta da ausencia do sol no inverno, então por este motivo foram criados, máquinas que aquecem o ambiente, parecido com um arcodicionado, porém, mas potente para manter uma enorme estufa aquecida para os frutos. Só foi possível criar esta máquina, graças a criação do arcodicionado, pois este aquecedor é uma melhoria do mesmo arcodicionado utilizado em escritórios. Assim como os termometros, graças as termometros para medir a temperatura do corpo, dos recipie ntes, dos alimentos, do ambiente, etc, foi criado uma versão melhorada para medir a temperatura do ambiente da estufa. Gertalmente os próprios gerados tem esse termometro, mas ele só mede a temperatura que este está liberando e não a medida da temperatura do ambiente por inteiro da estufa, por isso foi criado esta melhoria especialmente para as estufas. Foram criados também sistemas de irrigações, onde são instaladas várias torneiras automáticas onde são programadas para cada determinado momento as torneiras se abrirem e irregar as plantas automaticamente.//
